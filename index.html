Dron Tello edu

Tello EDU es un dron impresionante y programable perfecto para la educación. Puede aprender fácilmente lenguajes de programación como Scratch, Python y Swift. Con un SDK 2.0 actualizado, Tello EDU viene con comandos más avanzados y mayores interfaces de datos. Completo con la tecnología de control de vuelo de DJI, Tello EDU también admite Estabilización Electrónica de Imagen. Escriba código para ordenar a varias EDU Tello que vuelen en un enjambre y desarrolle sorprendentes funciones de IA. ¡Programar nunca ha sido tan divertido con Tello EDU!

Robobloq Qoopers

Qoopers es un robot muy versátil y expresivo que permite profundizar en el aprendizaje y conocimiento de la programación y la robótica. Cuenta con sensores y actuadores cuyo funcionamiento puede ser comandado con cualquier dispositivo móvil y PC, utilizando programación gráfica con bloques icónicos o con bloques escritos, programación en código Python o Arduino IDE. Qoopers cuenta con una placa Arduino adaptada con 8 puertos de sencilla conexión para sensores y actuadores que pueden estar operando en simultáneo. La placa, llamada Q-mind plus permite expandir sus posibilidades y conectarse a una Raspberry Pi y/o una placa Micro:bit.

Brazo robotico con arduino

Este proyecto tiene como objetivo la creación de un brazo robótico versátil y asequible, diseñado para ser controlado de forma intuitiva mediante sensores infrarrojos y una placa Arduino. Este brazo robótico no solo representa una emocionante incursión en el mundo de la robótica, sino que también puede ser una herramienta educativa valiosa y encontrar aplicaciones prácticas en la automatización de tareas simples.
